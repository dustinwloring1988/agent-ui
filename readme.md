## Quick Start

### System Requirements

- **macOS**: 10.15 or higher
- **Windows**: Windows 10 or higher
- **Linux**: Ubuntu 18.04+ / Debian 10+ / Fedora 32+
- **Memory**: 4GB+ recommended
- **Storage**: 500MB+ available space

### Install

<p>
  <a href="https://github.com/dustinwloring1988/agent-ui/releases">

</p>

```bash
# macOS via Homebrew
brew install aionui
```
